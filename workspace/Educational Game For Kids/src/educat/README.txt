Tester.java is the main entry into the program

The user needs to have installed the Java Runtime Environment. This program was created with Java 8. Any JRE past Java 8 will be sufficent. To install the Java runtime visit https://www.java.com/en/download/help/download_options.xml
where you can follow the instructions for your operating system. 
Once you do this and the Java Runtime has been installed you can run the program from the Tester.java class by right clicking and choosing run by administrator
